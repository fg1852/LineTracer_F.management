# LineTracer_F.management

정해진 경로로 이동해 창고로 물품을 운반하는 라인 트레이서

창고에 여유가 없으면 자동으로 다음 창고를 목적지로 지정

중앙 제어 시스템으로 창고 상태 확인, 라인트레이서의 이동중인 경로 표시.
